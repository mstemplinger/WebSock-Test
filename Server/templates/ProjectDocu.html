Server & Client Interaktion

🛠 Überblick

Dieses System besteht aus einem zentralen Server, der mit mehreren Clients über WebSockets und HTTP-API-Endpunkte kommuniziert.
Der Server empfängt, speichert und verarbeitet Daten von den Clients und ermöglicht zentrale Steuerungs- und Automatisierungsprozesse.

🌍 Server-Funktionalität

Der Server übernimmt folgende Hauptaufgaben:

1️⃣ Verwaltung von Clients

  • Registriert und speichert verbundene Clients
  • Aktualisiert Verbindungsstatus und letzte Aktivitäten
  • Listet verbundene Clients in einer Web-Oberfläche

2️⃣ Empfang und Speicherung von Daten

  • Clients können Daten (z. B. Systeminformationen, Logs) an den Server senden
  • Diese Daten werden in einer SQL-Datenbank gespeichert
  • Eine Inbox-Funktionalität erlaubt eine spätere Verarbeitung

3️⃣ Zentrale Steuerung und Kommunikation

  • Admins können über den Server Nachrichten an Clients senden
  • Der Server kann Skripte an Clients verteilen und deren Ausführung überwachen
  • Steuerung und Verwaltung erfolgt über eine Web-Oberfläche oder API

4️⃣ WebSocket-Kommunikation für Echtzeit-Updates

  • Clients melden sich über WebSockets am Server an
  • Der Server sendet Steuerungsbefehle an Clients (z. B. Skriptausführung)
  • Clients bestätigen die erfolgreiche Verarbeitung von Aufgaben

5️⃣ Datenbankbasierte Steuerung

  • Alle empfangenen Daten und Interaktionen werden protokolliert
  • Clients können Konfigurationsänderungen und Befehle aus der Datenbank abrufen

🔄 Interaktion zwischen Server & Clients

1️⃣ Client-Registrierung

  • Beim Start meldet sich jeder Client automatisch beim Server an.
  • Der Client übermittelt:
    • Client-ID (eindeutig pro Gerät)
    • Hostname (Rechnername)
    • IP-Adresse (Netzwerkadresse)
  • Der Server speichert diese Informationen in der Datenbank und zeigt sie in der Web-Oberfläche an.

2️⃣ Echtzeit-Kommunikation über WebSockets

  • Nach der Registrierung bleibt der Client über WebSockets verbunden.
  • Der Server kann Befehle oder Skripte an Clients senden.
  • Der Client kann Statusmeldungen, Log-Dateien oder Systeminformationen zurücksenden.

3️⃣ Skriptausführung auf Clients

  • Admins können Skripte (z. B. PowerShell, Python, Bash) zentral verwalten.
  • Diese Skripte können an einzelne oder alle Clients gesendet und dort ausgeführt werden.
  • Der Server überwacht die Übertragung und den Erfolg der Skriptausführung.

Skriptverteilung Ablauf

  1. Admin lädt ein Skript in den Server-Skriptordner hoch.
  2. Über die Web-Oberfläche oder API wählt der Admin:
    • Client, an den das Skript gesendet werden soll
    • Skripttyp (PowerShell, Batch, Python, Shell)
  3. Der Server sendet das Skript in mehreren Chunks an den Client.
  4. Der Client führt das Skript aus und sendet das Ergebnis zurück.

4️⃣ Zentrale Steuerung per Web-Oberfläche oder API

Der Server stellt eine Web-Oberfläche zur Verwaltung der Clients bereit:

  • Client-Liste anzeigen
  • Client-Details abrufen
  • Nachrichten an Clients senden
  • Skripte an Clients verteilen
  • Gespeicherte Daten einsehen

Alternativ können Admins die REST-API nutzen, um Anfragen direkt zu stellen.

5️⃣ Inbox-Funktion – Datenübertragung & Speicherung

Clients können strukturierte Daten (JSON-Format) an den Server senden.
Diese Daten werden zunächst ungeprüft in der Inbox gespeichert und später verarbeitet.

Ablauf einer Inbox-Datenverarbeitung

  1. Client sendet eine JSON-Nachricht mit Daten an den Server.
  2. Der Server speichert diese in der Datenbank (Inbox-Tabelle).
  3. Ein Hintergrundprozess verarbeitet die Daten und speichert sie in den finalen Tabellen.
  4. Falls eine Fehlermeldung auftritt, wird diese im Log hinterlegt.

📌 Einsatzmöglichkeiten:

  • Software-Update-Informationen speichern
  • Logs & Fehlerberichte von Clients sammeln
  • Systemstatus und Konfigurationsdaten abrufen

🚀 Beispiel-Abläufe

📌 Ein Client meldet sich beim Server an

  • Client startet und verbindet sich über WebSockets.
  • Server speichert den Client in der Datenbank und zeigt ihn in der Web-Oberfläche an.

📌 Ein Skript wird an einen Client gesendet

  • Admin wählt ein PowerShell-Skript aus und sendet es an den Client.
  • Der Client empfängt das Skript, führt es aus und sendet das Ergebnis zurück.

📌 Datenübertragung vom Client zum Server

  • Client sendet Systeminformationen (z. B. Festplattenstatus, CPU-Auslastung).
  • Der Server speichert die Daten und zeigt sie in der Web-Oberfläche an.

📌 Admin sendet eine Nachricht an alle Clients

  • Admin gibt eine Nachricht ein und klickt auf „Senden“.
  • Alle Clients empfangen die Nachricht und zeigen sie an.

📌 Anleitung zur Installation und Ausführung des Python-Skripts unter Windows

Dieses Skript verwendet Flask, SQLAlchemy, WebSockets und weitere Abhängigkeiten. Die folgenden Schritte führen dich durch die Installation und den Start unter Windows.

1️⃣ Voraussetzungen

Bevor du startest, stelle sicher, dass du Python 3.8+ auf deinem Windows-System installiert hast.

🔹 Überprüfung der Python-Installation
Öffne eine PowerShell oder Eingabeaufforderung (cmd) und gib ein:

python --version

Falls Python nicht installiert ist, kannst du es von python.org herunterladen.

2️⃣ Virtuelle Umgebung erstellen (empfohlen)

Es ist sinnvoll, eine virtuelle Umgebung zu nutzen, damit sich die Abhängigkeiten nicht mit anderen Projekten vermischen.

Öffne eine PowerShell oder Eingabeaufforderung (cmd) und navigiere in das Verzeichnis, wo das Skript liegt:

cd C:\Pfad\zum\Projekt

Erstelle eine neue virtuelle Umgebung:

python -m venv venv

Aktiviere die virtuelle Umgebung:

venv\Scripts\activate

💡 Hinweis: Falls du eine Fehlermeldung erhältst, dass das Skript blockiert ist, setze diesen Befehl:

Set-ExecutionPolicy Unrestricted -Scope Process

Nun kannst du die virtuelle Umgebung mit venv\Scripts\activate aktivieren.

3️⃣ Notwendige Abhängigkeiten installieren

Installiere nun alle benötigten Python-Pakete mit pip:

pip install flask flask-socketio flask-sqlalchemy websockets sqlalchemy

Falls eine MySQL-Datenbank verwendet wird, benötigst du zusätzlich:

pip install pymysql

Falls PostgreSQL verwendet wird, dann:

pip install psycopg2

Falls du dir nicht sicher bist, welche Abhängigkeiten fehlen, kannst du alle Pakete aus einer requirements.txt installieren:

pip install -r requirements.txt

4️⃣ Das Skript starten

Navigiere zum Skriptverzeichnis, falls du es noch nicht getan hast:

cd C:\Pfad\zum\Projekt

Starte das Skript mit:

python scriptname.py

Falls Flask standardmäßig auf Port 5001 laufen soll, kannst du es explizit so starten:

python scriptname.py --host=0.0.0.0 --port=5001

Falls du es im Hintergrund ausführen möchtest (PowerShell):

Start-Process python -ArgumentList "scriptname.py"

5️⃣ Web-Oberfläche & API testen

Sobald das Skript läuft, kannst du die Web-Oberfläche in deinem Browser öffnen:

🔗 http://localhost:5001/

Falls du die API testen willst, kannst du mit Postman oder curl eine Anfrage senden. Zum Beispiel:

def get_tables():
    """Gibt alle Tabellen in der Datenbank zurück."""
    try:
        logging.info("🔄 Lade Tabellen aus der Datenbank...")
        db.Model.metadata.reflect(bind=db.engine)
        tables = list(db.Model.metadata.tables.keys())
        logging.info(f"📋 Tabellen gefunden: {tables}")
        return tables
    except Exception as e:
        logging.error(f"❌ Fehler beim Abrufen der Tabellen: {str(e)}")
        return []


def get_column_lengths(table_name):
    """Liest die maximale Feldlängen der Spalten einer Tabelle aus."""
    try:
        logging.info(f"🔍 Prüfe Spaltenlängen für Tabelle `{table_name}`")

        # Metadaten aktualisieren
        db.Model.metadata.reflect(bind=db.engine, raiseerr=True)

        # Tabelle abrufen
        table = db.Model.metadata.tables.get(table_name)

        if table is None:
            logging.error(f"❌ Tabelle `{table_name}` nicht gefunden!")
            raise ValueError(f"❌ Tabelle `{table_name}` nicht gefunden.")

        logging.info(f"✅ Tabelle `{table_name}` gefunden. Starte Spaltenprüfung...")

        column_lengths = {}

        for column in table.columns:
            logging.info(f"ℹ️ Prüfe Feld `{column.name}`...")

            if isinstance(column.type, (String, NVARCHAR)) and column.type.length:
                column_lengths[column.name] = column.type.length
                logging.info(f"✅ Feld `{column.name}` hat eine max. Länge von {column.type.length}")

        logging.debug(f"📊 Erkannte Feldlängen für `{table_name}`: {column_lengths}")
        return column_lengths

    except Exception as e:
        logging.error(f"🚨 Fehler in `get_column_lengths({table_name})`: {str(e)}")
        raise


def truncate_values(column_values, column_lengths):
    """Kürzt die Werte basierend auf den maximalen Feldlängen."""
    logging.info(f"🔍 truncate_values `{column_values}`")
    for column, max_length in column_lengths.items():
        if column in column_values and isinstance(column_values[column], str):
            original_value = column_values[column]
            truncated_value = original_value[:max_length]
            column_values[column] = truncated_value
            if original_value != truncated_value:
                logging.warning(f"⚠️ Wert für `{column}` wurde von {len(original_value)} auf {max_length} Zeichen gekürzt!")
    return column_values

def process_inbox():
    """Verarbeitet alle noch nicht bearbeiteten Einträge in der Inbox-Tabelle."""
    logging.info("🟢 Starte `process_inbox`-Thread...")

    while True:
        with app.app_context():
            try:
                db.Model.metadata.reflect(bind=db.engine)  # 🔄 Metadaten synchronisieren
                logging.info(f"📋 Tabellen: {db.Model.metadata.tables.keys()}")

                logging.info("🛠️ Filtere nach acx_inbox_processing_state='pending'")
                pending_entries = Inbox.query.filter(Inbox.acx_inbox_processing_state == literal("pending")).all()

                if not pending_entries:
                    logging.info("✅ Keine neuen Einträge zum Verarbeiten.")
                else:
                    for entry in pending_entries:
                        logging.info(f"🔄 Verarbeite Inbox-ID: {entry.acx_inbox_id}")
                        entry.acx_inbox_processing_state = "running"
                        entry.acx_inbox_processing_start = datetime.now(timezone.utc)
                        db.session.commit()

                        try:
                            logging.info(f"📥 Lade JSON-Inhalt für Inbox-ID: {entry.acx_inbox_id}")
                            json_content = json.loads(entry.acx_inbox_content)
                            logging.debug(f"📄 JSON-Inhalt: {json_content}")

                            content_section = json_content.get("Content")
                            if not isinstance(content_section, dict):
                                raise ValueError("❌ `Content`-Bereich fehlt oder ist ungültig")

                            table_name = content_section.get("TableName", "").strip()
                            data_entries = content_section.get("Data", [])
                            mappings = content_section.get("FieldMappings", [])

                            if not table_name or not data_entries or not mappings:
                                raise ValueError("❌ Fehlende Daten oder Mappings im JSON")

                            column_lengths = get_column_lengths(table_name)

                            for record in data_entries:
                                column_values = {}
                                logging.info(f"📑 Verarbeite Datensatz: {record}")

                                for mapping in mappings:
                                    db_field = mapping.get("TargetField", "").strip()
                                    expression = mapping.get("Expression", "").strip()

                                    if not db_field:
                                        raise ValueError("❌ `TargetField` fehlt in Mappings")
                                    if not expression:
                                        raise ValueError(f"❌ `Expression` fehlt für {db_field}")

                                    if expression == "NewGUID()":
                                        column_values[db_field] = str(uuid.uuid4())
                                    elif expression.startswith("{") and expression.endswith("}"):
                                        json_field = expression.strip("{}")
                                        column_values[db_field] = record.get(json_field, "")
                                    else:
                                        column_values[db_field] = expression

                                    logging.debug(f"🔄 Mapping `{db_field}`: `{expression}` -> `{column_values[db_field]}`")

                                column_values = truncate_values(column_values, column_lengths)

                                insert_query = text(f"""
                                    INSERT INTO {table_name} ({', '.join(column_values.keys())}) 
                                    VALUES ({', '.join([f':{key}' for key in column_values.keys()])})
                                """)

                                logging.info(f"📥 SQL Insert Query: {insert_query}")
                                logging.debug(f"📊 Insert-Daten: {column_values}")

                                db.session.execute(insert_query, column_values)

                            db.session.commit()
                            entry.acx_inbox_processing_state = "success"
                            entry.acx_inbox_processing_end = datetime.now(timezone.utc)
                            entry.acx_inbox_processing_log = "Verarbeitung erfolgreich"
                            db.session.commit()
                            logging.info(f"✅ Verarbeitung für Inbox-ID {entry.acx_inbox_id} abgeschlossen!")

                        except SQLAlchemyError as e:
                            logging.error(f"❌ SQL-Fehler bei Inbox-ID {entry.acx_inbox_id}: {str(e)}")
                            db.session.rollback()
                            entry.acx_inbox_processing_state = "error"
                            entry.acx_inbox_processing_log = str(e)
                            db.session.commit()

                        except Exception as e:
                            logging.error(f"❌ Fehler bei Inbox-ID {entry.acx_inbox_id}: {str(e)}")
                            entry.acx_inbox_processing_state = "error"
                            entry.acx_inbox_processing_log = str(e)
                            db.session.commit()

            except Exception as e:
                logging.error(f"❌ Fehler in `process_inbox`: {str(e)}")

        time.sleep(10)  # Alle 10 Sekunden prüfen



def is_port_in_use(port):
    """Prüft, ob der angegebene Port bereits in Benutzung ist."""
    with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
        return s.connect_ex(("0.0.0.0", port)) == 0

def check_for_refresh():
    """Überprüft, ob sich die Client-Liste geändert hat, und sendet nur dann ein Refresh"""
    global previous_clients
    current_clients = {client_id: {"hostname": data["hostname"], "ip": data["ip"]} for client_id, data in clients.items()}

    if previous_clients != current_clients:  # 🔍 Vergleiche mit vorheriger Liste
        previous_clients = current_clients  # 🔄 Update der vorherigen Clients
        socketio.emit("refresh", to=None)
        logging.info("🔄 Client-Liste geändert, Refresh gesendet.")
        print("🔄 Client-Liste geändert, Refresh gesendet.")

@app.route("/table/<table_name>")
def table_data(table_name):
    """Zeigt alle Daten aus einer bestimmten Tabelle an."""

    # 🛠️ Prüfen, ob die Tabelle existiert
    db.Model.metadata.reflect(bind=db.engine)
    if table_name not in db.Model.metadata.tables:
        logging.error(f"❌ Fehler: Tabelle `{table_name}` existiert nicht in der Datenbank!")
        return f"Fehler: Tabelle `{table_name}` nicht gefunden!", 404

    logging.info(f"📊 Lade alle Daten aus `{table_name}`")

    try:
        table = db.Model.metadata.tables[table_name]
        query = table.select()  # 🔄 KEINE Filterung auf client_id oder asset_id

        with db.engine.connect() as connection:
            result = connection.execute(query).fetchall()

        # 📌 Korrektur: Spaltennamen aus dem Table-Objekt holen
        column_names = [column.name for column in table.columns]

        return render_template("table_data.html", table_name=table_name, data=result, columns=column_names)

    except Exception as e:
        logging.error(f"❌ Fehler beim Abrufen der Daten für `{table_name}`: {str(e)}")
        return f"Interner Serverfehler: {str(e)}", 500

@app.route("/inbox", methods=["POST"])
def inbox():
    """Empfängt JSON-Daten und speichert sie ungeprüft in die Inbox-Tabelle zur späteren Verarbeitung."""
    try:
        data = request.get_json()
        if not data:
            return jsonify({"error": "❌ Leere Anfrage erhalten"}), 400

        # ✅ JSON als Zeichenkette speichern
        json_content = json.dumps(data, ensure_ascii=False)

        # 📥 Neuen Inbox-Eintrag erstellen
        new_entry = Inbox(
            acx_inbox_name=data.get("MetaData", {}).get("Name", "Unbekannt"),
            acx_inbox_description=data.get("MetaData", {}).get("Description", "Keine Beschreibung"),
            acx_inbox_creator=data.get("MetaData", {}).get("Creator", "Unbekannt"),
            acx_inbox_vendor=data.get("MetaData", {}).get("Vendor", "Unbekannt"),
            acx_inbox_content_type=data.get("MetaData", {}).get("ContentType", "unknown"),
            acx_inbox_content=json_content,  # Ungeprüftes JSON speichern
        )

        db.session.add(new_entry)

                    }, ensure_ascii=False)

@app.route("/get_scripts", methods=["GET"])
def get_scripts():
    """Liefert eine Liste der verfügbaren Skripte im Skriptverzeichnis (inkl. Unterverzeichnisse 1. Ebene)"""
    allowed_extensions = {".ps1", ".bat", ".py", ".sh", ".txt"}
    ignored_dirs = {"_obsolete_"}

    if not os.path.exists(SCRIPT_DIR):
        return jsonify({"error": "Skriptverzeichnis nicht gefunden!"}), 500

    try:
        scripts = []

        # 🔍 Durchlaufe das Hauptverzeichnis
        for root, dirs, files in os.walk(SCRIPT_DIR):
            # Nur die erste Ebene der Unterverzeichnisse betrachten
            rel_root = os.path.relpath(root, SCRIPT_DIR)  # Relativer Pfad zum Root
            if rel_root in ignored_dirs:
                continue  # 🚫 Verzeichnis _obsolete_ überspringen

            # 🔄 Unterverzeichnisse nach erster Ebene abschneiden
            if os.path.dirname(rel_root):
                continue  # Verzeichnisse aus tieferen Ebenen ignorieren

            for file in files:
                ext = os.path.splitext(file)[1]
                if ext in allowed_extensions:
                    script_path = os.path.join(rel_root, file) if rel_root != "." else file  # 🎯 Pfad zusammensetzen
                    scripts.append({
                        "name": script_path.replace("\\", "/"),  # Konsistente Pfade für Windows/Linux
                        "type": "powershell" if ext == ".ps1" else
                                "bat" if ext == ".bat" else
                                "python" if ext == ".py" else
                                "linuxshell" if ext == ".sh" else
                                "text"
                    })

        return jsonify(scripts)

    except Exception as e:
        return jsonify({"error": f"Fehler beim Abrufen der Skripte: {str(e)}"}), 500

async def handle_client(websocket):
    """Verarbeitet eingehende WebSocket-Verbindungen von Clients mit erweitertem Logging"""
    try:
        client_ip = websocket.remote_address[0] if websocket.remote_address else "Unbekannt"
        logging.info(f"🔌 Neuer Client verbunden von {client_ip}")

        async for message in websocket:
            try:
                logging.info(f"📩 Eingehende Nachricht von {client_ip}: {message}")

                data = json.loads(message)
                action = data.get("action")

                if action == "register":
                    client_id = data.get("client_id")
                    hostname = data.get("hostname")
                    ip_address = data.get("ip")

                    # **Datenvalidierung**
                    if not client_id or not hostname or not ip_address:
                        logging.warning(f"⚠️ Ungültige Registrierungsdaten von {client_ip}: {data}")
                        await websocket.send(json.dumps({"status": "error", "message": "Invalid registration data"}))
                        continue

                    # **Client im lokalen Dictionary speichern**
                    clients[client_id] = {"websocket": websocket, "hostname": hostname, "ip": ip_address}
                    logging.info(f"📥 Neuer Client zwischengespeichert: {clients[client_id]}")

                    # **✅ Datenbank-Operationen für `acx_asset`**
                    with app.app_context():
                        try:
                            logging.info(f"🔎 Suche nach bestehendem Asset für Client {client_id}...")
                            existing_asset = Asset.query.filter_by(client_id=client_id).first()

                            if existing_asset:
                                logging.info(f"🔄 Update bestehendes Asset: {existing_asset.client_id} (Last Seen: {existing_asset.last_seen})")
                                existing_asset.last_seen = datetime.now()
                            else:
                                logging.info(f"🆕 Neues Asset wird erstellt für Client {client_id} ({hostname}, {ip_address})")
                                new_asset = Asset(client_id=client_id, hostname=hostname, ip_address=ip_address)
                                db.session.add(new_asset)

                            db.session.commit()
                            check_for_refresh()  # 🔄 Überprüfe Client-Änderungen für Refresh
                            logging.info(f"✅ Client {client_id} erfolgreich registriert oder aktualisiert in `acx_asset`")

                        except SQLAlchemyError as e:
                            db.session.rollback()
                            logging.error(f"❌ Datenbankfehler bei Registrierung von {client_id}: {str(e)}")
                            await websocket.send(json.dumps({"status": "error", "message": f"Database error: {str(e)}"}))
                            continue
                        except Exception as e:
                            db.session.rollback()
                            logging.error(f"❌ Unerwarteter Fehler bei DB-Operation für {client_id}: {str(e)}")
                            await websocket.send(json.dumps({"status": "error", "message": f"Unexpected error: {str(e)}"}))
                            continue

                    # **✅ Bestätigung an den Client senden**
                    response = json.dumps({"status": "registered"})
                    await websocket.send(response)
                    check_for_refresh()  # 🔄 Überprüfe Client-Änderungen für Refresh
                    logging.info(f"📤 Registrierungsbestätigung an {hostname} ({ip_address}) gesendet")

                else:
                    logging.warning(f"⚠️ Unbekannte Aktion von {client_ip}: {data}")
                    check_for_refresh()  # 🔄 Überprüfe Client-Änderungen für Refresh
                    await websocket.send(json.dumps({"status": "error", "message": "Unknown action"}))

            except json.JSONDecodeError:
                logging.error(f"🚨 Ungültiges JSON von {client_ip}: {message}")
                await websocket.send(json.dumps({"status": "error", "message": "Invalid JSON"}))
            except Exception as e:
                logging.error(f"❌ Allgemeiner Fehler in der Nachricht von {client_ip}: {str(e)}")
                await websocket.send(json.dumps({"status": "error", "message": f"Processing error: {str(e)}"}))

    except websockets.exceptions.ConnectionClosed as e:
        logging.info(f"❌ WebSocket-Verbindung mit {client_ip} geschlossen: {e}")

    finally:
        # **Client aus der DB entfernen, wenn Verbindung verloren geht**
        with app.app_context():
            disconnected_clients = [key for key, value in clients.items() if value["websocket"] == websocket]

            for client_id in disconnected_clients:
                try:
                    logging.info(f"🚪 Entferne Client {client_id} aus Clientspeicher...")
                    clients.pop(client_id, None)

                    logging.info(f"🔎 Suche nach Asset {client_id} in `acx_asset` zur Löschung...")
                    asset = Asset.query.filter_by(client_id=client_id).first()

                    if asset:
                        logging.info(f"🗑️ Lösche Asset {client_id} aus `acx_asset`")
                        # db.session.delete(asset)
                        # db.session.commit()
                        logging.info(f"✅ Client {client_id} erfolgreich aus `acx_asset` entfernt.")
                    else:
                        logging.warning(f"⚠️ Kein Asset-Eintrag für {client_id} gefunden. Kein Löschvorgang durchgeführt.")

                except SQLAlchemyError as e:
                    db.session.rollback()
                    logging.error(f"❌ Fehler beim Löschen von Client {client_id} aus `acx_asset`: {str(e)}")
                except Exception as e:
                    logging.error(f"❌ Allgemeiner Fehler beim Entfernen von Client {client_id}: {str(e)}")

        check_for_refresh()  # 🔄 Überprüfe Client-Änderungen für Refresh
        logging.info("🛑 Client-Entfernung abgeschlossen.")

function Connect-WebSocket {

            $websocket = New-Object System.Net.WebSockets.ClientWebSocket

            Write-Log "🔄 Versuche Verbindung zum Server..."
            $websocket.ConnectAsync($uri, [System.Threading.CancellationToken]::None).Wait()


            Write-Log "✅ Erfolgreich verbunden!"

            $response = Register-Client -websocket $websocket

            } else {

        }

function Register-Client {
    param ([System.Net.WebSockets.ClientWebSocket]$websocket)

    $data = @{
        action    = "register"
        client_id = $clientId
        hostname  = $computerName
        ip        = $ipAddress
    } | ConvertTo-Json -Compress

    Write-Log "📤 Sende Registrierungsdaten: $data"
    $buffer = [System.Text.Encoding]::UTF8.GetBytes($data)
    $segment = New-Object System.ArraySegment[byte] -ArgumentList (, $buffer)

    $websocket.SendAsync($segment, [System.Net.WebSockets.WebSocketMessageType]::Text, $true, [System.Threading.CancellationToken]::None).Wait()

    $receiveBuffer = New-Object byte[] 8192
    $segment = New-Object System.ArraySegment[byte] -ArgumentList (, $receiveBuffer)
    $result = $websocket.ReceiveAsync($segment, [System.Threading.CancellationToken]::None).Result

    if ($result.Count -eq 0) {
        Write-Log "⚠️ Server hat eine leere Antwort gesendet!"
        return "error"
    }

    $message = [System.Text.Encoding]::UTF8.GetString($receiveBuffer, 0, $result.Count)
    Write-Log "📨 Server-Antwort: $message"

    $responseData = $message | ConvertFrom-Json -ErrorAction Stop

    if ($responseData.status -eq "registered") {
        return "registered"
    } else {
        Write-Log "⚠️ Unerwartete Server-Antwort: $message"
        return "error"
    }
}

function Process-IncomingChunk {
    param ([hashtable]$data)

    $scriptName = $data["script_name"]
    $chunkIndex = $data["chunk_index"]
    $totalChunks = $data["total_chunks"]
    $scriptChunk = $data["script_chunk"]
    $scriptType = $data["script_type"]  # 🆕 Neuer Parameter für Skripttyp

    Write-Log "📥 Empfangenes Skript-Chunk $chunkIndex/$totalChunks für $scriptName (Typ: $scriptType)"

    if (-not $scriptChunks.ContainsKey($scriptName)) {
        $scriptChunks[$scriptName] = @{}
        $scriptTotalChunks[$scriptName] = $totalChunks
    }

    $scriptChunks[$scriptName][$chunkIndex] = $scriptChunk
    $receivedChunksCount = $scriptChunks[$scriptName].Count

    Write-Log "🧐 Chunks für $scriptName - $receivedChunksCount / $totalChunks"

    if ($receivedChunksCount -eq $totalChunks) {
        Write-Log "✅ Alle Chunks für $scriptName empfangen. Setze Datei zusammen..."

        $fullScriptBase64 = ""
        for ($i = 0; $i -lt $totalChunks; $i++) {
            if ($scriptChunks[$scriptName].ContainsKey($i)) {
                $fullScriptBase64 += $scriptChunks[$scriptName][$i]
            } else {
                Write-Log "❌ Fehler: Chunk $i fehlt! Abbruch."
                return
            }
        }

        try {
            # Skript dekodieren (Base64 → UTF-8)
            $decodedBytes = [System.Convert]::FromBase64String($fullScriptBase64)
            $scriptContent = [System.Text.Encoding]::UTF8.GetString($decodedBytes)

            if ($scriptType -eq "powershell-base64") {
                Write-Log "▶️ Direktes Ausführen des Base64-codierten PowerShell-Skripts im Speicher"

                # Skript direkt in PowerShell übergeben und ausführen
                $encodedScript = [Convert]::ToBase64String([System.Text.Encoding]::Unicode.GetBytes($scriptContent))
                Start-Process -FilePath "powershell.exe" -ArgumentList "-ExecutionPolicy Bypass -EncodedCommand $encodedScript" -NoNewWindow -Wait
                Write-Log "✅ PowerShell-Base64-Skript ausgeführt"

            } else {
                # Datei speichern und ausführen
                $timestamp = Get-Date -Format "yyyyMMdd_HHmmss"
                $safeScriptName = $scriptName -replace '[^\w\.-]', '_'
                $tempFile = "$ScriptDir\$timestamp`_$safeScriptName"

                if (-Not (Test-Path $ScriptDir)) {
                    New-Item -ItemType Directory -Path $ScriptDir -Force | Out-Null
                }

                Write-Log "💾 Speichere vollständiges Skript in: $tempFile"
                $scriptContent | Out-File -Encoding utf8 $tempFile

                if (-Not (Test-Path $tempFile)) {
                    Write-Log "❌ Fehler: Datei $tempFile wurde nicht gespeichert!"
                    return
                }

                Write-Log "📄 Datei erfolgreich gespeichert: $tempFile"
                Write-Log "▶️ Starte Skript: $tempFile"

                # Passenden Interpreter wählen
                switch ($scriptType) {
                    "powershell" {
                        Start-Process -FilePath "powershell.exe" -ArgumentList "-ExecutionPolicy Bypass -File `"$tempFile`"" -NoNewWindow 
                    }
                    "bat" {
                        Start-Process -FilePath "cmd.exe" -ArgumentList "/c `"$tempFile`"" -NoNewWindow 
                    }
                    "python" {
                        Start-Process -FilePath "python" -ArgumentList "`"$tempFile`"" -NoNewWindow 
                    }
                    "linuxshell" {
                        Start-Process -FilePath "bash" -ArgumentList "`"$tempFile`"" -NoNewWindow 
                    }
                    "text" {
                        Write-Log "📄 Öffne Skript als Textdatei in Notepad: $tempFile"
                        Start-Process -FilePath "notepad.exe" -ArgumentList "`"$tempFile`""
                    }
                    default {
                        Write-Log "⚠️ Unbekannter Skripttyp: $scriptType. Standard: PowerShell"
                        Start-Process -FilePath "powershell.exe" -ArgumentList "-ExecutionPolicy Bypass -File `"$tempFile`"" -NoNewWindow 
                    }
                }

                Write-Log "✅ Skript erfolgreich ausgeführt: $scriptName"
            }

            # Speicher für das Skript leeren
            $scriptChunks.Remove($scriptName)
            $scriptTotalChunks.Remove($scriptName)

        } catch {
            Write-Log "❌ Fehler beim Dekodieren oder Speichern des Skripts: $_"
        }
    }
}



function Listen-WebSocket {
    param ([System.Net.WebSockets.ClientWebSocket]$websocket)

    while ($true) {
        if ($websocket -eq $null -or $websocket.State -ne "Open") {
            Write-Log "⚠️ Verbindung verloren. Versuche eine neue Verbindung..."
            Start-Sleep -Seconds 5
            $websocket.Dispose()
            $websocket = Connect-WebSocket
            continue
        }

        try {
            Write-Log "⏳ Warte auf Nachricht..."
            $receiveBuffer = New-Object byte[] 8192
            $segment = New-Object System.ArraySegment[byte] -ArgumentList (, $receiveBuffer)
            $result = $websocket.ReceiveAsync($segment, [System.Threading.CancellationToken]::None).Result

            if ($result.MessageType -eq "Close") {
                Write-Log "⚠️ Verbindung vom Server geschlossen."
                Start-Sleep -Seconds 5

                # WebSocket-Client starten
        $websocket = Connect-WebSocket -serverUrl $serverUrl

                continue
            }

            # Nachricht dekodieren
            $message = [System.Text.Encoding]::UTF8.GetString($receiveBuffer, 0, $result.Count)
            Write-Log "📨 Nachricht erhalten: $message"

            # JSON-Parsing mit Fehlerbehandlung
            try {
                $data = $message | ConvertFrom-Json -ErrorAction Stop
            } catch {
                Write-Log "⚠️ Ungültige JSON-Nachricht: $message"
                $data = $null
            }

            # Falls die Nachricht JSON enthält, prüfen wir die "action"
            if ($data -ne $null -and $data.PSObject.Properties["action"]) {
                switch ($data.action) {
                    "message" {
                        if ($data.content -eq "STOP") {
                            Write-Log "🛑 STOP-Nachricht erhalten. Beende das Skript..."
                            $websocket.CloseAsync([System.Net.WebSockets.WebSocketCloseStatus]::NormalClosure, "STOP received", [System.Threading.CancellationToken]::None).Wait()

                            if ($psISE) {
                                Write-Log "⚠️ Skript läuft in PowerShell ISE. Schließe das ISE-Fenster nicht automatisch."
                                exit
                            } else {
                                Write-Log "💨 Skript wird jetzt geschlossen..."
                                Stop-Process -Id $PID  # Beendet das PowerShell-Fenster
                            }
                        } else {
                            Write-Log "📩 Nachricht vom Server: $($data.content)"
                            Start-Process -FilePath "msg.exe" -ArgumentList "* $($data.content)"
                        }
                    }
                    "upload_script_chunk" {
                        $hashtableData = @{}
                        $data.PSObject.Properties | ForEach-Object { $hashtableData[$_.Name] = $_.Value }
                        Process-IncomingChunk -data $hashtableData
                    }
                    default {
                        Write-Log "⚠️ Unbekannte Aktion: $($data.action)"
                    }
                }
            } else {
                Write-Log "⚠️ Nachricht kein gültiges JSON oder kein 'action'-Feld: $message"
            }

        } catch {
            Write-Log "❌ Fehler beim Empfangen der Nachricht: $_"
        }
    }
}

import asyncio
import websockets
import json
import os
import socket
import uuid
import base64
import subprocess

# 🌍 WebSocket-Server URL
SERVER_URL = "ws://85.215.147.108:8765"

# 📡 Client-Identifikationsdaten
CLIENT_ID = str(uuid.uuid4())
COMPUTER_NAME = socket.gethostname()
IP_ADDRESS = socket.gethostbyname(socket.gethostname())

# 📁 Speicherort für Skripte und Logs
SCRIPT_DIR = "/tmp/scriptfiles"
LOG_FILE = "/tmp/client_stream.log"

# 🔄 Skript-Chunks speichern
script_chunks = {}
script_total_chunks = {}
script_types = {}

# 📁 Sicherstellen, dass das Skriptverzeichnis existiert
os.makedirs(SCRIPT_DIR, exist_ok=True)


# 📝 Logging-Funktion
def log_message(message):
    timestamp = asyncio.get_event_loop().time()
    log_entry = f"{timestamp:.2f} - {message}"
    print(log_entry)
    with open(LOG_FILE, "a") as log_file:
        log_file.write(log_entry + "\n")


# 🛠 Funktion zur `dos2unix`-Konvertierung
def convert_to_unix(script_path):
    try:
        result = subprocess.run(["dos2unix", script_path], check=True, capture_output=True, text=True)
        log_message(f"🔄 `dos2unix` erfolgreich ausgeführt für: {script_path}")
    except FileNotFoundError:
        log_message("⚠️ `dos2unix` ist nicht installiert! Installiere es mit `sudo apt install dos2unix`.")
    except subprocess.CalledProcessError as e:
        log_message(f"❌ Fehler bei `dos2unix`: {e.stderr}")


# 🔄 Funktion zur Skriptausführung
def execute_script(script_name, script_content_base64, script_type):
    script_dir = SCRIPT_DIR
    os.makedirs(script_dir, exist_ok=True)  # Sicherstellen, dass das Verzeichnis existiert

    # 🔽 Base64-Dekodierung
    try:
        script_content = base64.b64decode(script_content_base64).decode("utf-8")
    except Exception as e:
        log_message(f"❌ Fehler beim Dekodieren des Skripts: {e}")
        return

    # 🔽 Korrekte Speicherung des Skripts
    safe_script_name = script_name.replace("/", "_")  # Sonderzeichen entfernen
    script_path = os.path.join(script_dir, safe_script_name)

    try:
        with open(script_path, "w") as file:
            file.write(script_content)

        os.chmod(script_path, 0o755)  # Datei ausführbar machen
        log_message(f"💾 Skript gespeichert: {script_path}")

        # 🔄 `dos2unix` ausführen (Windows-Zeilenumbrüche entfernen)
        convert_to_unix(script_path)

        # 🔽 Skript basierend auf dem Typ ausführen
        if script_type == "linuxshell":
            subprocess.run(["/bin/bash", script_path], check=True)
        elif script_type == "python":
            subprocess.run(["python3", script_path], check=True)
        elif script_type == "text":
            print(open(script_path).read())
        else:
            log_message(f"⚠️ Unbekannter Skripttyp: {script_type}")

        log_message(f"✅ Skript erfolgreich ausgeführt: {script_name}")

    except Exception as e:
        log_message(f"❌ Fehler beim Speichern oder Ausführen des Skripts: {e}")


# 📡 Client am WebSocket-Server registrieren
async def register_client(websocket):
    data = {
        "action": "register",
        "client_id": CLIENT_ID,
        "hostname": COMPUTER_NAME,
        "ip": IP_ADDRESS
    }
    await websocket.send(json.dumps(data))
    log_message(f"📤 Sende Registrierungsdaten: {data}")


# 🔄 Chunks verarbeiten
def process_chunk(script_name, chunk_index, total_chunks, script_chunk, script_type):
    global script_chunks, script_total_chunks, script_types  # Verwenden der globalen Variablen

    log_message(f"📥 Chunk {chunk_index}/{total_chunks} für {script_name} erhalten")

    # Stelle sicher, dass script_chunks[script_name] als Dictionary existiert
    if script_name not in script_chunks:
        script_chunks[script_name] = {}
        script_total_chunks[script_name] = total_chunks
        script_types[script_name] = script_type  # Speichert den Typ für spätere Verarbeitung

    script_chunks[script_name][chunk_index] = script_chunk  # Chunk abspeichern

    # Prüfen, ob alle Chunks vorhanden sind
    received_chunks = len(script_chunks[script_name])
    log_message(f"🔄 Chunks empfangen: {received_chunks} / {total_chunks}")

    if received_chunks == total_chunks:
        log_message(f"✅ Alle Chunks für {script_name} empfangen. Setze Datei zusammen...")

        # Chunks in der richtigen Reihenfolge zusammensetzen
        sorted_chunks = [script_chunks[script_name][i] for i in range(total_chunks)]
        full_script_base64 = "".join(sorted_chunks)

        # Führe das Skript aus
        execute_script(script_name, full_script_base64, script_types[script_name])

        # Speicher bereinigen
        del script_chunks[script_name]
        del script_total_chunks[script_name]
        del script_types[script_name]


# 🔄 WebSocket-Verbindung aufbauen & Nachrichten verarbeiten
async def connect_websocket():
    while True:
        try:
            log_message("🔄 Versuche Verbindung zum Server...")
            async with websockets.connect(SERVER_URL) as websocket:
                log_message("✅ Erfolgreich verbunden!")
                await register_client(websocket)

                async for message in websocket:
                    log_message(f"📨 Nachricht erhalten: {message}")

                    # Prüfen, ob gültiges JSON
                    try:
                        data = json.loads(message)
                    except json.JSONDecodeError:
                        log_message("⚠️ Fehler beim JSON-Parsen, wird ignoriert.")
                        continue

                    action = data.get("action")

                    if action == "register":
                        await register_client(websocket)

                    elif action == "upload_script_chunk":
                        process_chunk(
                            data.get("script_name"),
                            data.get("chunk_index"),
                            data.get("total_chunks"),
                            data.get("script_chunk"),
                            data.get("script_type")
                        )

                    elif action == "message":
                        content = data.get("content")
                        if content == "STOP":
                            log_message("🛑 STOP-Befehl erhalten. Beende das Skript...")
                            await websocket.close()
                            os._exit(0)
                        else:
                            log_message(f"📩 Nachricht vom Server: {content}")

                    else:
                        log_message(f"⚠️ Unbekannte Aktion: {action}")

        except Exception as e:
            log_message(f"⚠️ Verbindung verloren: {e}. Neuer Versuch in 5 Sekunden...")
            await asyncio.sleep(5)


# 🔥 WebSocket starten
if __name__ == "__main__":
    asyncio.run(connect_websocket())
